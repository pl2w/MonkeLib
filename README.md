# MonkeLib
A library mod made for Gorilla Tag mod development.

## Contributing
Feel free to contribute anything that would ease mod development!
Make sure your pull request is well written and clear on what it does.

## Features

Skin material indexes as an enum.<br>
RoomSystemWrapping for easier effect networking.
